Sessão de um processo

A Sessão de um processo, é em qual sessão este processo foi iniciado.

No caso de sistemas operacionais Windows, quando um usuário loga no sistema, uma sessão é criada, desta maneira, o Windows organiza os processos e serviçõs que cada usuário abre em uma sessão separada, como se fosse um "container".

Quando o sistema inicia, a sessão 0 é iniciada e utilizada pelos processos e serviços essenciais para o funcionamento do sistema operacional, após isso, o primeiro usuário a logar no sistema é colocado na sessão 1, o segundo na sessão 2, e assim por diante.


Neste caso, todos os processos com a sessão "Services" estão na sessão 0, e os outros com a sessão "Console" por um usuário, que pode ser identificado pelo seu nome na coluna "Usuário"
