Encerrando um processo

Para encerrar o processo, rodamos o comando 'taskkill /PID pidDoProcesso' de acordo com o processo selecionado, no caso, pidDoProcesso é substituido pelo PID do processo desejado.

Perceba que nem todo processo é possível encerrar, isso pois o sistema operacional nos impede de encerrar processos críticos para seu funcionamento.

Perceba também que, esta aplicação também é um processo!. Procure pelo processo com o nome de java.exe e tente encerrá-lo!
