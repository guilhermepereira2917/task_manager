Argumentos de um processo

Os argumentos de um processo são os argumentos opcionais que são passados para um processo, por exemplo, o comando tasklist /V /NH /FO csv possui três argumentos opcionais, como mencionado anteriormente: /V, /NH e /FO csv.

Outro exemplo seria algum comando do tipo: ping www.google.com -n 10, que pinga www.google.com 10 vezes.

Porém, por algum motivo, o Java dificilmente encontra os argumentos de um processo, mas mesmo assim, deixamos o campo aqui de qualquer maneira.
