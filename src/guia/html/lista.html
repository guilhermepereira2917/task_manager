Lista de processos

Aqui está uma lista de processos, para mostrá-la na tela, o Java basicamente roda o comando 'tasklist /V /NH /FO csv' e consome seu conteúdo para exibí-lo na tela.

Explicando o comando em partes: tasklist é o comando em si, que pede ao Windows quais processos estão rodando neste exato momento, /V é o parametro que pede ao comando para ser verboso, no caso, apresentar mais opções do que ele normalmente apresentaria (como Estado do Processo e tempo de CPU) /NH é um parametro que pede para que o cabeçalho de identificação das colunas não seja mostrado, e o parametro /FO csv pede para que o comando seja formatado no padrão CSV, para ficar mais fácil de consumir seu conteúdo.

Este comando é chamado uma vez a cada segundo por uma Thread, ao mesmo tempo que permite com que seja interagido com a interface.


Clique no cabeçalho das colunas para saber mais sobre cada um dos campos.

Clique com o botão direito em um processo para encerrá-lo ou visualizar suas propriedades, ou tentar, pelo menos.
